# RAWDA Production, Sales & Operations Performance Dashboard

## Overview

This project presents an interactive Power BI dashboard developed to analyze production, sales, and operational performance. The dashboard helps identify key trends, monitor KPIs, and support data-driven decision-making.

## Tools Used

* Power BI
* Microsoft Excel
* Data Cleaning and Transformation

## Dashboard Pages

1. Cover Page
2. Production Analysis
3. Sales Analysis
4. Operations Analysis
5. Insights and Recommendations

## Key Insights

* 5 Gallon was the top-performing product.
* Line B recorded the highest production output.
* Abu Dhabi emerged as the strongest-performing region.
* Retail was the leading sales channel.
* Machine downtime affected operational efficiency.

## Recommendations

* Implement preventive maintenance to reduce downtime.
* Focus production planning on high-demand products.
* Improve inventory and logistics planning.
* Strengthen underperforming sales channels.
* Enhance demand forecasting using historical trends.
